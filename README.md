# Instructions

1. Clone the repo
```
$ git clone https://github.com/ujaiahmad/flutter_movie_app.git
```
2. Change directory
```
$ cd flutter_movie_app
```
3.Fetch Dependencies from pubspec.yaml
```
$ flutter pub get
```
4. Run the App
```
$ flutter run
```
